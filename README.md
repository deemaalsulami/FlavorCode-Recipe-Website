# FlavorCode-Recipe-Website
An interactive recipe website that helps users discover recipes, explore ingredients, and use a step-by-step Cook Mode experience. Built with HTML, CSS, and JavaScript.

## Overview
FlavorCode is an interactive recipe website designed to help users discover recipes, explore ingredients, and follow cooking instructions through a step-by-step Cook Mode experience.

## Features
- Recipe browsing and discovery
- Recipe categories and filtering
- Search functionality
- Interactive Cook Mode
- Ingredient tracking
- Cooking timer
- User profile section
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
